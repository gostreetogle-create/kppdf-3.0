# kp-product-picker — модалка выбора товаров/услуг

> Компонент: `kp-product-picker.component.ts` · используется в **QuotationEditor**

## Назначение

Крупная модалка «витрина» для подбора позиций из справочника `products` в таблицы документа (КП и др.).

## UX (актуально с 2026-05-28)

| Элемент | Поведение |
|---------|-----------|
| **Выбор строк** | Клик по **всей строке** таблицы (не только по чекбоксу); крупный индикатор выбора ~22px |
| **Пагинация** | PrimeNG **Paginator** (номера страниц, first/last, «Показано 1–25 из N») |
| **Корзина-чипы** | **Убраны** — выбранные позиции видны по подсветке строк и счётчику «Выбрано: N» |
| **Подтверждение** | Закреплённый **футер диалога**: слева сводка, справа **«Добавить N товаров»** (крупная primary-кнопка) |
| **Подсказка** | В шапке модалки при выборе: «нажмите «Добавить …» внизу окна» |

## Фильтры по типу таблицы

Родитель передаёт `[defaultFilters]` — для таблицы **Товары** фильтр `kind: ITEM`, для **Услуги** — `kind: SERVICE`.

## API (inputs / outputs)

| Input | Описание |
|-------|----------|
| `visible` | двусторонняя привязка открытия |
| `multiple` | мультивыбор (true) / одна позиция (false) |
| `defaultFilters` | начальные фильтры (search, categoryId, kind, activeOnly) |
| `selectedIds` | id уже добавленных в документ (строки disabled) |
| `confirmLabel` | база подписи кнопки (по умолчанию «Добавить») |

| Output | Описание |
|--------|----------|
| `productsSelected` | массив выбранных `IProduct` (multi) |
| `productSelected` | одна позиция (single / dblclick) |
