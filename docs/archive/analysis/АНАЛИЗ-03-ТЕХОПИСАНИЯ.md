# АНАЛИЗ-03: Папка «Тех описание изделий» (Технические описания)

## Выявленные сущности, поля, связи и паттерны

---

## 📄 Файл 1: 4.html — Описание

### Сущность: TechSpec (Техническое описание)
HTML-страница с подробным описанием конструкции изделия. Содержит:
- **Раздел 1**: общее описание, назначение
- **Раздел 2**: технические характеристики (материалы, профили, размеры сечения)
- **Раздел 3**: чертежи и схемы (в виде изображений)
- **Раздел 4**: требования к монтажу и установке
- **Раздел 5**: условия эксплуатации

Характерные элементы описания:
- Используемые профили: 120х120х4, 80х80х3, 40х40х2 и т.д.
- Тип покрытия: порошковая окраска, метод запекания
- Класс защиты: IP54 (для электрооборудования)
- Способ монтажа: бетонирование, анкерный

---

## 📄 Файл 2: И.П.Желонкин.html — Техописание по поставщику

### Сущность: SupplierSpec (Спецификация поставщика)
Описание продукции, поставляемой ИП Желонкин. Содержит:
- Перечень изделий с техническими характеристиками
- Условия изготовления
- Гарантийные обязательства

---

## 📄 Файл 3: Описание.html — Общее описание

### Сущность: ProductDescription (Описание продукта)
- Типовое описание конструкции уличных тренажёров и спортоборудования
- Стандартные формулировки: 
  - «Изделие, устанавливается в местах общественного пользования...»
  - «Окрашено порошковой краской методом запекания...»
  - «Предназначено для тренировки мышц...»

---

## 📄 Файл 4: Лист4.html — Технический лист

### Сущность: TechSheet (Технический лист)
- Дополнительные технические характеристики
- Схемы сборки
- Спецификация комплектующих

---

## 📄 Файл 5: МК.html — Металлоконструкции

### Сущность: MetalStructure (Металлоконструкция)
- Описание сварных металлоконструкций
- Типы соединений (сварные, болтовые, хомутовые)
- Толщина металла по ГОСТ

---

## 🧩 Итоговая сущность

```
TechDescription (Техническое описание)
├── id
├── product_id (связь с Product)
├── title (название раздела/документа)
├── content (HTML-содержимое с разметкой)
├── images[] (ссылки на изображения/чертежи)
├── type (enum: 'general', 'supplier', 'tech_sheet', 'metal_work')
│
├── MaterialSpec (Спецификация материалов)
│   ├── profile_type (труба профильная/круглая/лист)
│   ├── dimensions (120х120х4мм)
│   └── quantity (количество)
│
└── InstallationGuide (Инструкция по монтажу)
    ├── method (бетонирование/анкерный)
    ├── requirements (требования к основанию)
    └── safety_notes (меры безопасности)
```
