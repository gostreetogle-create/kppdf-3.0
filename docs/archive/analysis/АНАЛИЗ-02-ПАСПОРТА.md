## 📄 Файл 1: pasports.html — Реестр паспортов изделий

### Сущность: ProductPassport (Паспорт изделия)

| Поле | Тип | Пример |
|------|-----|--------|
| `passport_number` | number | 224–275 (84 записи) |
| `date` | date | 15.08.2025 – 11.10.2025 |
| `warranty_code` | string | 20258-224, 202510-275 (формат: ГГГГМ-№) |
| `product_code` | number | 2254–2307 (внутренний код изделия) |
| `photo` | image | Ссылка на фото каждой единицы |
| `category` | string | Категория изделия |
| `name` | string | Наименование |
| `article` | string? | Артикул (не у всех, опционально) |
| `height` | number | Высота в мм |
| `length` | number | Длина в мм |
| `width` | number | Ширина в мм |
| `weight` | number | Вес в кг |
| `description` | text | Полное техописание изделия |
| `installation_site` | string | Объект установки |
| `supplier` | string? | Поставщик |

### Категории изделий (из реестра паспортов)

| Категория | Кол-во | Примеры |
|-----------|--------|---------|
| Спортивное оборудование | ~25 | Стойки баскетбольные, ворота, флагштоки |
| Уличный тренажёр | ~20 | Жим, скороход, велотренажёр |
| Воркаут | ~15 | Турники, брусья, скамьи, площадки |
| Благоустройство | ~15 | Скамьи, секции ограждения, урны |
| тренажёр (без подкатегории) | ~10 | Разные тренажёры |

### Паттерн: Даты и нумерация
- Дата → формат ГГГГМ-№: 20258 = август 2025, 20259 = сентябрь 2025, 202510 = октябрь 2025
- Паспорта нумеруются сквозным счётчиком (224–275)
- Внутренний код изделия (2254–2307) — сквозной по всем типам продукции

### Паттерн: Описание изделия
Каждое описание следует шаблону:
```
[Название] в дальнейшем именуемые как 'Изделие', изготовленная из [материалы].
Изделие окрашено порошковой краской методом запекания в заводских условиях.
Изделие, устанавливается в [место установки].
Предназначено для [назначение].
```

## 📄 Файл 2: Products.html — Каталог продукции

### Сущность: Product (Товар/Изделие) — расширенная версия

| Поле | Тип | Описание |
|------|-----|----------|
| `article` | string? | Артикул |
| `category` | string | Категория |
| `name` | string | Наименование |
| `height` | number | Высота (мм) |
| `length` | number | Длина (мм) |
| `width` | number | Ширина (мм) |
| `weight` | number | Вес (кг) |
| `materials` | text | Изготовленная из... (материалы, профили, трубы) |
| `installation` | text | Изделие устанавливается... (место установки) |
| `purpose` | text | Предназначено для... (назначение) |
| `full_description` | text | Полный текст описания |

### Разделение спецификации
В products.html описание разбито на 3 отдельных поля:
1. **materials** — «изготовленная из труб 80х80х3мм... окрашено порошковой краской...»
2. **installation** — «Изделие устанавливается в местах общественного пользования...»
3. **purpose** — «Предназначено для игры в баскетбол»

### Паттерн: Материалы (часто используемые профили)
| Материал | Описание |
|----------|----------|
| Труба профильная | 120х120х4мм, 80х80х2мм, 40х40х2мм, 20х20х2мм |
| Труба круглая | ∅89х3мм, ∅48х2.8, ∅33.5х2.8 |
| Лист стальной | 2, 3, 4, 6, 8 мм |
| Дерево | Брус 100х100мм (воркаут), доска 40х40мм (скамьи) |
| Поликарбонат | Монолитный для баскетбольных щитов |
| Покрытие | Порошковая краска, запекание в заводских условиях |

## 🧩 Итоговые сущности (для БД)

```
Product (Изделие/Товар)
├── id, article, category_id, name
├── height, length, width, weight
├── materials (текст)
├── installation (текст)
├── purpose (текст)
├── description (текст — полный)
├── image (ссылка)
│
├── ProductPassport (Паспорт изделия)
│   ├── id, product_id
│   ├── passport_number, date, warranty_code
│   ├── product_code, article
│   ├── photo
│   ├── installation_site
│   │
│   └── ProductCategory (Категория)
│       └── id, name, description
```

### Ключевые связи
```
Product 1──1 ProductPassport
ProductCategory 1──N Product
Product → Category (категория как справочник)
```

### Отличия от АНАЛИЗ-01-КП.md
В products.html (КП) товары имеют **цену**, но нет размеров/веса/материалов.
В Products.html (Паспорта) — наоборот: есть **размеры/вес/материалы**, но нет цены.
→ **Это разные проекции одной сущности Product**

| Проекция | Откуда | Поля |
|----------|--------|------|
| Коммерческая | КП/products.html | Название, категория, цена, ед. изм. |
| Техническая | Pasports/Products.html | Габариты, вес, материалы, установка |
