## 📄 Файл 1: products.html — Прайс-лист (каталог товаров)

### Сущность: Product (Товар/Изделие)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | number | № п/п (1–34+) |
| `category` | string | Категория: «Стела», «Информационный стенд», «Благоустройство», «Остекление», «Стенд» |
| `name` | string | Наименование товара |
| `specs` | text | Характеристики изделия (подробное описание: материалы, размеры, технология) |
| `unit` | string | Ед. изм.: «шт.» или «м2» |
| `price` | number | Цена за ед. (руб) |
| `image` | image | Фото товара (для некоторых позиций) |

### Характеристики товара (specs) — паттерн
Каждая характеристика — это текст, содержащий:
- **Габариты** (Д×Ш×В в мм)
- **Материалы** (сталь, нержавейка, дерево, чугун, стекло)
- **Технологии** (порошковая окраска, УФ-печать, пескоструйка)
- **Электрооборудование** (подсветка 12-24В, блок питания, IP54)
- **Метод монтажа** (бетонирование, анкерный)

### Категории товаров (из таблицы)
| Категория | Кол-во позиций | Диапазон цен |
|-----------|---------------|--------------|
| Стела | 3 | 160,000 – 540,000 |
| Информационный стенд | 1 | 290,000 |
| Стенд | 2 | 360,000 – 476,000 |
| Благоустройство | 22+ | 33,000 – 4,700,000 |
| Остекление | 4 | 6,250/м2 |

### Паттерны
- **Цена = фиксированная за единицу** (нет скидок, нет наценок)
- **Категория как группа** — для фильтрации
- **Изображение** прикреплено к товару (Google Sheets inline image)

| Поле | Тип | Пример |

| `product_name` | string | Велопарковка... |
| `quantity` | number | 1 |
| `unit` | string | шт. |
| `attachments` | string | «Таблица 1» |
| `delivery_terms` | text | «Стоимость включает изготовление, упаковку, доставку, установку» |
| `response_requirements` | text | «КП с указанием НМЦК...» |
| `legal_basis` | text | «Федеральный закон № 44-ФЗ...» |

### Связи (очень важные!)
```
Tender.company_id → Company.id
Tender.product_name → Product.name (по наименованию, нестрогая)
Tender.product_id → Product.id (будет в идеале)
```

### Паттерн: Типовые условия
У всех 27 запросов **одинаковые** поля:
- `delivery_terms` — идентичный текст
- `response_requirements` — идентичный текст
- `legal_basis` — одинаковый (44-ФЗ)
- **Это можно вынести в шаблон/справочник**

### Паттерн: Дата
- Все 27 запросов датированы **22.05.2026** (один день)
- Вероятно, дата = дата регистрации входящего пакета

## 🧩 Итоговые сущности (для БД)

Из папки КП выделены следующие бизнес-сущности:

```
Company (Юрлицо-участник)
├── id, name, email, supplier_email
│
├── Tender (Запрос/Тендер)
│   ├── tender_id, number, date, company_id
│   ├── subject, product_name, quantity, unit
│   ├── delivery_terms (шаблон)
│   ├── response_requirements (шаблон)
│   ├── legal_basis (справочник)
│   └── attachments
│
└── Product (Товар/Изделие) — из прайс-листа
    ├── id, category, name, specs, unit, price, image
    └── category → справочник категорий
```

### Ключевые связи
```
Company 1──N Tender
Product 1──N Tender (через product_name или product_id)
Product.category → Category
Tender.delivery_terms → Template (шаблон)
Tender.legal_basis → LawReference (44-ФЗ)
```
