## 📊 ПАТТЕРН 2: Счётчик номеров

**Источник:** setings.html, mail_data.html, pasports.html

Система нумерации в бизнесе:
| Тип | Формат | Пример | Описание |
|-----|--------|--------|----------|
| Тендер | XXXX_NN | 5202_94 | 5202 = общий код, NN = номер внутри диапазона |
| Паспорт | ГГГГМ-№ | 20258-224 | 20258 = август 2025, № = сквозной |
| Гарантийный | ГГГГМ-№ | 20258-224 | = номеру паспорта |
| Код изделия | число | 2254 | Сквозной для всех изделий |

**Вывод:** В проекте уже есть Counters → нужно добавить форматы для Tender и Passport.
